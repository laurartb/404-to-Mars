# 404 to Mars
Hackatón Web, Página 404. Laura Rojo Trueba (Udit, 2025).

Este proyecto es una página de error 404 responsive diseñada con una estética futurista inspirada en terminales digitales y pantallas de advertencia. 
Combina tipografía digital, colores negro y verde neón y animaciones inspiradas en fallos de sistema.
Incluye una pantalla de carga con efecto glitch, un texto con animación de escritura estilo consola, un efecto neón con vibración tipo monitor antiguo y un diseño responsive (clamp).
